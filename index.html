Proyectos

Python

Este proyecto demuestra mis conocimientos en Programación Orientada a Objetos (POO) utilizando Python. Construí una pequeña aplicación para gestionar una biblioteca de libros, donde se pueden agregar, eliminar y listar libros usando clases y objetos, con herencia y polimorfismo.

C++

Utilicé programación orientada a objetos para estructurar el juego en clases como Jugador, Enemigo y Objeto.
